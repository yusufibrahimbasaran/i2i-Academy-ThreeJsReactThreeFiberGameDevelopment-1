# Three.js / React Three Fiber Game

A simple 3D endless runner game built with React Three Fiber for the i2i Academy internship task. 

In this game, you control a blue box and try to dodge incoming red obstacles. The score increases the longer you survive.

## Features
- Built with React Three Fiber and Three.js
- Simple keyboard controls (A/D or Arrow Keys)
- Dynamic obstacle spawning
- Basic collision detection
- Score tracking and Game Over screen

## How to run locally
1. `npm install`
2. `npm run dev`
3. Go to `http://localhost:5173`
